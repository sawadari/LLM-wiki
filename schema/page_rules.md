# Page Rules

## 共通ルール

- ページ名は短く、意味が分かるものにする
- 1ページ1主題を基本とする
- 関連ページへのリンクをできるだけ持たせる
- 断定には根拠を伴わせる

## 推奨見出し

### entity page

- Summary
- Role
- Related Concepts
- Related Sources
- Open Questions

### concept page

- Definition
- Compared Concepts
- Importance
- Limits
- Related Pages

### analysis page

- Question
- Assumptions
- Comparison Scope
- Findings
- Reservations
- Next Questions

### comparison page

- Compared Items
- Comparison Axes
- Differences
- Suitable Conditions
- Unsuitable Conditions
